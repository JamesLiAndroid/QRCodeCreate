# QRCodeCreate

The QRCode generate tools created by php

二维码生成工具：
1. 文字输入生成二维码
2. 图片上传存储后生成图片完整路径对应的二维码
3. 移动端图片拍照或者选择图片上传，存储后生成图片完整路径对应的二维码
